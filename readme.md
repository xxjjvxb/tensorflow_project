

# Tensorflow 로 작성하는 딥러닝 프로젝트
영상처리와 관련된 딥러닝 논문들의 모델들을 구현한다. 모델들의 특징을 이해하고 간단한 설명을 Markdow을 통하여 소개한다. 논문 구현 언어는 Python 3.5+를 이용하고 딥러닝 프레임워크로는 Tensorflow를 이용한다. 그 외 pandas, numpy, scipy 등의 라이브러리를 이용한다.
